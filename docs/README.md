# Relatos Documentation

## Specifications

Relatos uses the **relat** text format for data. The following specs are available in this directory:

| Document | Description |
|----------|-------------|
| [RELAT-SPEC.md](./RELAT-SPEC.md) | relat text format: syntax, nodes, edges, groups, style, comments |
| [RELAT-EXPORT-SPEC.md](./RELAT-EXPORT-SPEC.md) | Export rules: how `exportRelat` outputs relat text |
| [RELAT-STYLE-SPEC.md](./RELAT-STYLE-SPEC.md) | Style block: `style { node/group/edge/rel ... }` |
| [RELAT-LAYOUT-SPEC.md](./RELAT-LAYOUT-SPEC.md) | Layout block: `layout { node/group/edge ... }` |
| [GRAPH-EDIT-SPEC.md](./GRAPH-EDIT-SPEC.md) | Graph edit mode: move/resize nodes and groups, edge anchors and bends |
